# Contributing

Thank you for your interest in contributing to this project! Here's how you can get involved.

## Reporting Bugs & Requesting Features

The best way to contribute is by opening an issue:

- **Found a bug?** Open a [bug report](https://github.com/magaransoft/typed-map/issues/new?template=bug_report.md) with steps to reproduce.
- **Have an idea?** Open a [feature request](https://github.com/magaransoft/typed-map/issues/new?template=feature_request.md) describing the use case.

## Code Contributions

If you'd like to contribute code, please start by opening an issue to discuss the change. This helps ensure the work aligns with the project's direction and avoids duplicated effort.

Once we've discussed the approach, I may invite you to submit a pull request. This keeps the process collaborative and makes sure contributions are a good fit for the project.

## Questions

If you have a question about usage or behavior, feel free to open an issue — no template needed.
